# ТЗ: Edge Profiles — Приложение Kiln Calculator

## Контекст

Kiln Calculator — приложение для расчёта загрузки печей и конфигурации продукта.
Нужно добавить выбор профиля торца как часть конфигурации продукта.
Эти данные потом используются Sales при создании заказа.

---

## Что нужно сделать

### 1. Справочник профилей торцов

Добавить справочник (таблица или константы):

| Код | Название | Иконка/Схема | Описание |
|-----|----------|------------|----------|
| `straight` | Прямой | `▯` | Стандартный 90° торец (по умолчанию) |
| `beveled_45` | Фаска 45° | `◣` | Срез под 45 градусов |
| `beveled_30` | Фаска 30° | `◣` | Срез под 30 градусов (более пологий) |
| `rounded` | Скруглённый | `◠` | Скруглённый край (radius) |
| `bullnose` | Полукруг | `⌢` | Полный полукруг (180°) |
| `pencil` | Карандаш | `·` | Маленькое скругление на краю |
| `ogee` | Ogee | `~` | S-образный профиль |
| `waterfall` | Водопад | `⌐` | Скос + скругление |
| `stepped` | Ступенчатый | `⌐` | Уступ на торце |
| `custom` | Кастомный | `?` | Произвольный (описание текстом) |

### 2. UI: Выбор профиля в конфигураторе продукта

Где: в карточке продукта, рядом с полями Size, Shape, Thickness.

**Поле: "Профиль торца" (Edge Profile)**
- Тип: визуальный selector (иконки профилей + название)
- По умолчанию: `straight`
- При наведении/клике — показывать схему среза торца

**Поле: "Кол-во сторон" (Sides)**
- Показывается когда выбран не `straight`
- Визуальное отображение: прямоугольник плитки, клик на сторону включает/выключает обработку
- Или простые кнопки: 1 / 2 / 3 / 4
- По умолчанию: 1

**Поле: "Примечание" (Notes)**
- Только для `custom`
- Текстовое поле, 255 символов

### 3. Влияние на расчёт загрузки печи

**Профиль торца НЕ влияет на:**
- Физические размеры плитки (L × W × thickness)
- Вместимость печи
- Решение flat vs edge loading
- Расчёт количества рядов/колонок

**Профиль торца ВЛИЯЕТ на:**
- Площадь глазурования торца (для расчёта расхода материалов)
- Это считается на стороне PMS, не в Kiln Calculator

Поэтому в Kiln Calculator **не нужно менять формулы** — только добавить UI для выбора профиля и передачу данных.

### 4. Хранение и передача данных

Данные edge profile сохраняются как часть конфигурации продукта:

```json
{
  "product_config": {
    "color": "Moss Glaze",
    "size": "10x10",
    "shape": "rectangle",
    "thickness_mm": 14,
    "finishing": "Matt",
    "edge_profile": "bullnose",
    "edge_profile_sides": 2,
    "edge_profile_notes": null
  }
}
```

Эти данные передаются в Sales App при формировании заказа (Sales включает их в webhook payload для PMS).

### 5. Визуализация на схеме печи

Если в батче есть плитки с нестандартными торцами — показывать маркер:
- Оранжевый badge на плитке в схеме загрузки
- Tooltip: "Bullnose ×2"

Это опционально — можно реализовать позже.

---

## Допустимые значения (enum)

Строго фиксированный набор. Kiln Calculator, Sales и PMS должны использовать **одинаковые коды**:

```
straight, beveled_45, beveled_30, rounded, bullnose, pencil, ogee, waterfall, stepped, custom
```

Не добавлять новые значения без синхронизации с PMS.

---

## Порядок реализации

1. Добавить справочник профилей (с визуальными иконками)
2. Добавить поля в конфигуратор продукта
3. Сохранять edge_profile в конфигурации
4. Передавать в Sales при экспорте/создании заказа

---

## Backward Compatibility

- Существующие конфигурации без edge_profile → считаются `straight`
- Kiln Calculator можно обновлять независимо от PMS и Sales
